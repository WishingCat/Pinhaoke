# 北大公选课数据说明

## 数据来源

北京大学选课系统 `elective.pku.edu.cn`，2025-2026学年第2学期公选课，共 **256 门课程**。

## 文件格式

JSON 数组，每个元素是一门课程，包含 `基本信息` 和 `详细信息` 两个对象。

```json
{
  "基本信息": {
    "课程号": "02535760",
    "课程名": "非物质文化遗产创意与实践",
    "课程类别": "全校任选、劳动教育课",
    "学分": "1.0",
    "教师": "锁凌燕(教授)",
    "班号": "1",
    "开课单位": "经济学院",
    "专业": "",
    "年级": "",
    "上课时间及教室": "",
    "限数已选": "20 / 0",
    "自选PNP": "不可申请",
    "备注": ""
  },
  "详细信息": {
    "英文名称": "Creative Practice of Intangible Cultural Heritage",
    "先修课程": "无先修要求",
    "中文简介": "本课程是一门融合...",
    "英文简介": "This is a distinctive course...",
    "成绩记载方式": "合格制",
    "通识课所属系列": "",
    "授课语言": "中文",
    "教材": "",
    "参考书": "",
    "教学大纲": "通过代表性非遗传承人的讲解...",
    "教学评估": "锁凌燕：学年度学期：25-26-2..."
  }
}
```

## 字段说明

### 基本信息

| 字段 | 类型 | 填充率 | 说明 |
|------|------|--------|------|
| 课程号 | string | 100% | 8位数字编码，同一门课不同班号共享课程号 |
| 课程名 | string | 100% | |
| 课程类别 | string | 100% | 如"全校任选""全校任选、劳动教育课""全校任选、思政选择性必修" |
| 学分 | string | 100% | 数值型字符串，如"2.0" |
| 教师 | string | 66% | 格式为"姓名(职称)"，多位教师用逗号分隔；部分课程（主要是清华课程）此字段为空 |
| 班号 | string | 100% | |
| 开课单位 | string | 100% | |
| 专业 | string | 0% | 公选课无此限制，全部为空 |
| 年级 | string | 7% | 少数课程限定年级，如"2023""2024" |
| 上课时间及教室 | string | 97% | 格式如"1~15周 每周周二10~11节 二教210"，少数无固定时间的慕课为空 |
| 限数已选 | string | 100% | 格式为"限数 / 已选人数"，如"50 / 0" |
| 自选PNP | string | 100% | "可申请"或"不可申请" |
| 备注 | string | 66% | 补充说明，如"清华课程；...""本研合上""Taught in English"等 |

### 详细信息

| 字段 | 类型 | 填充率 | 说明 |
|------|------|--------|------|
| 英文名称 | string | 100% | |
| 先修课程 | string | 100% | 多数为"无先修要求" |
| 中文简介 | string | 100% | 课程内容介绍，长度从几十字到上千字不等 |
| 英文简介 | string | 100% | |
| 成绩记载方式 | string | 100% | 如"百分制""等级制""合格制" |
| 通识课所属系列 | string | <1% | 公选课几乎不属于通识课系列 |
| 授课语言 | string | 100% | 如"中文""英文" |
| 教材 | string | 0% | 均为空 |
| 参考书 | string | 0% | 均为空 |
| 教学大纲 | string | 100% | 课程目标、教学内容等 |
| 教学评估 | string | 65% | 包含教师评估信息，格式为"教师名：学年度学期：...，课程推荐得分：..." |

## 注意事项

- 所有字段均为字符串类型，学分等数值需自行转换
- 同一课程号可能出现多条记录（不同班号），如"30300945 制造工程体验"有多个班
- "限数已选"中的已选人数为抓取时刻的快照（2026年3月20日）
- 教材和参考书字段在系统中未填写，全部为空
